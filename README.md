# 2019-nCoV数据项目作业要求

2019-nCoV数据的采集、存储、**处理**、**展示**。

## 具体要求

1. 有需求分析
   - 用处
   - 好处
2. 数据来源不限
3. 数据内容形式不限
   - 数字
   - 图片
   - 文字
4. 开发语言不限（*"为了更好地展示所学内容", Yan*）
5. 开发平台不限
   - Web
   - 移动应用
   - 小程序（便于大家查询）
   - VR眼镜
   - 全息投影
6. 展示形式
   - 图片
   - 热力图
   - 分布情况
   - 柱形图
   - 饼图
   - 折线图
   - 多维
7. 竞品分析（现有工具的优缺点）
   - 现有展示形式
   - 现有应用工具

## 提交内容

- 源代码
- 答辩 PPT
- 说明文档
  - 设计动机
  - 概要设计
  - 详细设计
  - 展示形式
  - 展望
- 必须**全英文 Full English**

## 评分标准

- 数据采集、存储、处理、展示的合理性
- 课程结合程度
  - 什么数据结构
  - 原因
  - 目的
- 团队分工合作
  - 需求分析
  - 前端展示
  - 后端处理
  - 数据库结构
  - 写文档
  - 答辩
  - 会议/讨论记录
  - 组长
    - 统筹安排
    - 解决分歧
- 小组答辩
  - 中英皆可，**英文有加分**
  - **对别组提问有加分**
